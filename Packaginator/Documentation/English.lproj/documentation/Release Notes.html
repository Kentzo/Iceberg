Release Notes

Version 1.2
  • General:

    • New: Universal Binary

  • IcebergBuilder:

    • New: Command line tool to be able to split forks without Developer Tools installed.
    • New: Ability to clean .svn folders.
    • Fix: IFRequirements.strings are now correctly saved with Unicode Encoding.

  • Iceberg:

    • New: Build window to get more information on the build process and potential problems.
    • New: Support for Installer plugins.
    • New: Updated Preferences Window with more options and new layout.
    • New: Quickpick project template
    • Enh: Import Requirements from PackageMaker XCode 2.2 format.
    • Enh: Better import support for incomplete PackageMaker project file.
    • Fix: "Internet Plug-ins" is actually "Internet Plug-Ins".
    • Fix: Selecting a language from the action pop-up menu now correctly enables the OK button.
    • Fix: The Delete... menu item for the Additional Resources contextual menu was disabled if the file was MIA.
    • Chg: The IFPkgFlagInstallFat flag is no more supported in the User Interface as it's not supported by Mac OS X.

  • freeze:

    • IMPORTANT: The freeze tool has been moved from /usr/bin/ to /usr/local/bin/.

Version 1.1.6
  • IcebergBuilder:

    • Fix: Only one Title and one Message were created during a component build.
    • Fix: The .lproj folders were not created if needed for Requirements strings.

  • Iceberg:

    • New: Reference Style support for the Imported Package archive files.
    • New: Integrated Czech localization.
    • Mod: Disable unknown and nobody user and group in the File Inspector as the cpio format can't handle them currently.
    • Fix: '==' was still used instead of '=' in the "doesn't exist" scenario for bundle and file requirements.
    • Enh: Support for Drag and Drop for the Preference Pane, Spotlight Importer and Screen Saver templates plug-in.

Version 1.1.5
  • IcebergBuilder:

    • New: Add a IFPkgCreator key to know which tool and which version built a component.
    • New: Support for the IFPkgFlagInstalledSize flag.

    • Enh: Improved support for build made on a remote volume.

    • Fix: FolderInfo flags are now correctly set on real folder whose content has been disclosed.

  • Iceberg:

    • Enh: About box: The version field does not display any highlight anymore and clicking on it does not stop the Credits animation anymore.
    • Fix: Project window: Build status text field is now resizable.
    • Fix: Documents pane: Fix a typo.
    • Fix: Documents pane: Complete missing French localizations.
    • New: Scripts pane: Support for Gestalt IFRequirement.
    • Fix: Scripts pane: Use = instead of == for the requirements operator. == does not work on 10.4.x.
    • Fix: Scripts pane: French localization was lacking the Reference Style column in the Installation Scripts table.
    • New: Inspector and Files pane: It's now possible to modify the files/folders permissions for a multiple selection (unless the selection includes a non modifiable item).
    • Fix: Inspector and Files pane: Fix a bug in the display of the file permissions when the sticky bit is set and the executable bit for others is not set.
    • Fix: Inspector and Files pane: Fix a bug with multiple selection in the Inspector.
    • Fix: Inspector and Files pane: The default reference style shall be selected even if it's not part of the defaults yet.
    • Fix: Inspector and Files pane: Disabling/Enabling a Search Rule was removing the attributes.
    • New: Project Templates: Spotlight Importer Template Project.
    • Enh: Project Templates: Take into account the new sticky bit on /Library on Tiger.
    • Enh: Project Templates: Updated PreferencePane and ScreenSaver template project.
    • New: Components Import: Can import .pmproj projects (Distribution Script format is not supported yet).
    • New: Components Import: Import Metapackages (recursively or not depending on preferences).
    • Enh: Components Import: Preserve IFPkgFlagInstalledSize flag.
    • Enh: Components Import: Preserve IFPkgPathMappings flag.
    • Enh: Components Import: Improve the reading of .info files not created by PackageMaker or Iceberg.
    • Fix: Components Import: Import background.tiff background picture.
    • New: Project Assistant: New Plugin architecture to support additional Project Assistant panes.
    • Upd: Simulator: Use defaultBackground.tif instead of background.tif for the default background picture.
    • Enh: Localizable values: When you switch from a language to another one, the currently edited value is now saved.
    • New: License Templates: CeCILL 2 license (English and French).
    • New: Help Menu: Documentation is available via the Help menu.

  • Distribution:

    • Upd: Completely redesigned and updated documentation.
    • New: Documentation is now available in French.
