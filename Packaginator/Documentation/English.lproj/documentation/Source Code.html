Source Code

The source code for most parts of Iceberg is available under the BSD license (unless stated otherwise in the source file):

  • Projects Type: XCode 2.3
  • Language: Objective-C, C, CoreFoundation, Cocoa
  • Available there: Source code
The source code archive contains the following projects:

Name  Description 
Packaginator Project for the the front-end Iceberg application.
IcebergBuilder Project for the tool building Iceberg projects.
Iceberg Control Tower    Project for the daemon used to launch the building of projects.
freeze Project for the command line tool used to launch a build from the Terminal. 

Some parts are not in the archive for distribution reasons, you will need to download them separately:
  • MOKit Framework: mokit.sourceforge.net
