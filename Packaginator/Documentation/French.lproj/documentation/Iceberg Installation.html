Installation

Ce document décrit comment installer Iceberg à partir de l'image disque que vous avez téléchargée ou récupérée depuis un support.

Iceberg étant un outil destiné à fabriquer des packages et metapackages, l'installateur d'Iceberg est tout naturellement un package crée avec Iceberg.

Configuration requise pour l'installation

  • Mac PowerPC ou Intel.

  • Mac OS X 10.2 ou ultérieur.

  • L'installation requiert un utilisateur possédant des privilèges administrateurs. Si vous n'êtes pas administrateur sur la machine que vous utilisez, contactez votre administrateur technique.

Installation locale sur un Macintosh

1Double-cliquez sur l'image disque Iceberg.
2Double-cliquez sur le volume Iceberg.
3Double-cliquez sur le package Iceberg.pkg afin de démarrer l'installation.
4

Suivez les instructions qui s'affichent dans l'application Programme d'installation.

Note : Lors de l'installation proprement dite des fichiers, le Programme d'installation recherchera la présence d'une version antérieure d'Iceberg. S'il s'agit de la première installation d'Iceberg sur votre Macintosh, l'opération de recherche peut prendre un petit peu de temps et l'application Iceberg sera alors installée par défaut dans le dossier /Applications.

Installation distante sur un ou plusieurs Macintosh

Iceberg étant distribué sous la forme d'un package, il est possible de l'installer sur un ordinateur distant. En fonction de la version de Mac OS X que vous possèdez, il existe deux solutions pour des installations distantes :

Installation avec l'outil en ligne de commande installer

1

Copiez le package Iceberg.pkg sur la machine distante.

Note : Si aucun service de partage de fichiers (FTP, AppleShare, etc.) n'est activé sur la machine distante, vous devrez vous déplacer pour réaliser la copie et l'installation.
2Connectez vous sur la machine distante (via ssh ou telnet).
3Placez vous dans le répertoire où se trouve le package Iceberg.pkg sur la machine distante.
4

Tapez la commande suivante :

$ installer -pkg Iceberg.pkg -target /

Installation avec Apple Remote Desktop

Apple Remote Desktop (ARD) est une solution de gestion de parc informatique pour Mac OS X qui permet entre autres d'installer à distance des packages.

Consultez le chapitre 3 page 58 de la documentation (en Anglais) de ce logiciel. L'installation distante de package avec ARD y est expliquée en détail.

Topographie de l'installation

L'installation du package Iceberg.pkg copie divers composants à de multiples emplacements. Le tableau ci-dessous montre quel composants vont être installés et à quel emplacement.

Emplacement  Composant  Description 
/Applications/ Iceberg.app Application principale qui permet de créer des packages et metapackages.
/Library/Application Support/   Iceberg Dossier qui contient les modèles de licence utilisateur et les modèles de projet.
/Library/Frameworks/ MOKit.framework Framework utilisé par l'application Iceberg.
/Library/StartupItems/ IcebergControlTower   StartupItem qui gére les processus en tache de fond qui sont utilisés pour construire les projets.
/usr/local/bin/ freeze Outil qui permet de lancer des constructions de projet depuis la ligne de commande.
/usr/local/bin/ goldin Outil qui permet de séparer les forks d'un fichier depuis la ligne de commande.

Désinstallation

Si vous souhaitez désinstaller Iceberg de votre Macintosh, vous devez posséder des privilèges administrateurs.

1Double-cliquez sur l'image disque Iceberg.
2Double-cliquez sur le volume Iceberg.
3Lancez l'application Terminal.
4Tapez sudo sh dans la ligne de commande.
5

Glissez-déposez depuis le Finder le fichier uninstall.sh qui se trouve au premier niveau du volume Iceberg.

Exemple :
$ sudo sh /Volumes/Iceberg\ 1.1.4/uninstall.sh
6Tapez sur la touche Return pour valider la commande et saisissez votre mot de passe administrateur pour autoriser l'exécution de la commande.
