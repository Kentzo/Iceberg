Code source

Le code source de la majorité d'Iceberg est disponible sous licence BSD (dans le cas contraire, la licence est indiquée dans les fichiers sources) :

  • Type de projets : XCode 2.3
  • Language : Objective-C, C, CoreFoundation, Cocoa
  • Disponible ici : Code source
L'archive du code source contient les projets suivants :

Nom  Description 
Packaginator Projet de l'application Iceberg.
IcebergBuilder Projet de l'outil qui construit les projets Iceberg.
Iceberg Control Tower    Projet du daemon utilisé pour lancer la construction des projets Iceberg. 
freeze Projet de l'outil en ligne de commande.

Certaines parties ne sont pas présentes dans l'archive pour des raisons de distributions (et afin que vous puissiez utiliser les dernières versions disponibles de celles-ci). Vous devrez donc les télécharger séparément :
  • MOKit Framework: mokit.sourceforge.net
