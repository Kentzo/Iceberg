Création d'un package

Ce document décrit comment réaliser un package qui installe l'application TextEdit dans le dossier /Applications. Il permet de découvrir les panneaux Configurations et Fichiers.

Création du nouveau projet

1Lancez l'application Iceberg.
2Choisissez Fichier > Nouveau Projet… .
3Choisissez le modèle de projet Package et cliquez sur Suivant.
4Tapez MonPackage dans le champ Nom du projet et cliquez sur Terminer.

Ajout des fichiers à installer

1Choisissez Vue > Fichiers.
2Cliquez sur le triangle à gauche du dossier / afin de révéler son contenu.
3Sélectionnez la ligne Applications.
4Choisissez Archive > Ajouter des fichiers… .
5Naviguez jusqu'au dossier /Applications.
6Sélectionnez l'application TextEdit et cliquez sur Ajouter.
7Sélectionnez le style de référence à utiliser pour le(s) élément(s).

Note : Le style de référence par défaut peut être modifié dans le dialogue Préférences.

8Cliquez sur Ajouter.

Paramètrage des options du package

L'installation d'éléments dans le dossier /Applications nécessite que l'utilisateur ait des privilèges administrateur. Il faut donc que l'utilisateur ait à s'identifier comme administrateur de la machine lors de l'installation du package.

1Choisissez Vue > Configurations.
2Sélectionnez Autorisation root dans le menu local Autorisation de la section Options.

Construction du package

1Choisissez Fichier > Enregistrer.
2Choisissez Construction > Construire.

Le package MonPackage.pkg sera créé dans le dossier build qui se trouve au même niveau que le document du projet.
